## Installation

### Building from source

```bash
$ mkdir build/ && cd build/
$ cmake .. && make
```

## Usage

```bash
$ ./exray
```

If you want to convert the output PPM file, use [ImageMagick](https://imagemagick.org/index.php)
```bash
$ magick image.ppm image.png
```
